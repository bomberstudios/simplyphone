# SimplyPhone

A simple theme for jailbroken iPhones with WinterBoard.


## Features

- Simple, smooth icons.
- WeatherIcon support.
- LiveClock support.
- No icon labels.


## Installation

- Jailbreak your iPhone and install WinterBoard, WeatherIcon and LiveClock.
- Copy the theme folder to /Library/Themes/
- Enable the theme in WinterBoard.
